> **Purpose:** This document provides a comprehensive overview for AI agents and developers working with this codebase.

## Project Overview

**OS-APOW (Orchestrator for Sentinel, Planning, and Orchestration Workflows)** is an autonomous AI orchestration system that transforms GitHub Issues into execution orders fulfilled by specialized AI agents. The system is self-bootstrapping, meaning it can build and evolve itself using its own orchestration capabilities.

## Architecture

### Core Components

1. **Work Event Notifier ("The Ear")**
   - FastAPI webhook receiver at `src/orchestrator/services/notifier.py`
   - Handles GitHub events (issues, issue_comment, pull_request)
   - HMAC SHA256 validation for security
   - Transforms events into unified `WorkItem` objects

2. **Work Queue ("The Logic State")**
   - GitHub-backed queue using labels as state indicators
   - Implementation at `src/orchestrator/queue/github_queue.py`
   - `ITaskQueue` abstraction enables provider swapping (Linear, Jira, etc.)

3. **Sentinel Orchestrator ("The Brain")**
   - Persistent supervisor at `src/orchestrator/services/sentinel.py`
   - Polls for `agent:queued` issues every 60 seconds
   - Manages worker lifecycle via shell bridge
   - Posts heartbeat comments during long-running tasks

4. **Opencode Worker ("The Hands")**
   - Agentic execution environment
   - Reads workflow modules from `local_ai_instruction_modules/`

### State Machine (Label Logic)

| Label | State | Description |
|-------|-------|-------------|
| `agent:queued` | Awaiting | Task validated, awaiting Sentinel pickup |
| `agent:in-progress` | Active | Sentinel claimed task, execution underway |
| `agent:reconciling` | Recovery | Stale task detected, awaiting re-assignment |
| `agent:success` | Complete | Workflow finished successfully |
| `agent:error` | Failed | Technical failure, logs posted to issue |

- **Framework:** FastAPI (async)
- **Package Manager:** uv
- **Validation:** Pydantic v2

- **Linting:** ruff

## Entry Points

Defined in `pyproject.toml`:

```bash
uv run orchestrator  # Main FastAPI app (orchestrator.main:run)
uv run sentinel      # Sentinel service (orchestrator.services.sentinel:run)
uv run notifier      # Notifier service (orchestrator.services.notifier:run)
```
